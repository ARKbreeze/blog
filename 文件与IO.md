# 文件与IO

# 计算机多级体系

- cpu 三级缓存 寄存器 速度最快
- 内存 断电清除内容  速度中等
- 硬盘 速度最慢

# 文件的本质

- 文件的编码不同，解析方式不同，导致文件的表现不同

- 16进制转换成2进制特别方便     7  ->  0111 
- 如果不是对文件系统十分熟悉，请永远使用绝对路径

## 输入流与输出流 

- inputStream 输入流
  1. fileInputStream 文件输入流
- outputStream 输出流
  1. fileOutputStream 文件输出流
