-  http 状态码 
  1. 1** 服务器收到，需要客户端继续执行
  2. 2** 成功，操作成功并处理
  3. 3** 重定向，需要进一步操作以完成请求
  4. 4** 客户端错误，请求包含违法信息无法完成请求
  5. 5** 服务器错误，服务器在处理构成中出现错误
